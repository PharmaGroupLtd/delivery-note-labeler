Delivery Note Labeler - Install on another PC
=============================================

Requirements
  - Windows 10 or Windows 11 (64-bit)
  - No .NET, SDK, Visual Studio, or admin rights required

Install (recommended)
  1. Copy DeliveryNoteLabeler-1.0.1-Setup.exe to the target PC.
  2. Double-click it and follow the prompts.

Install (zip fallback)
  1. Extract DeliveryNoteLabeler-1.0.1-win-x64.zip.
  2. Double-click Install.cmd (or run Install.ps1 in PowerShell).

Use
  - App location:
      %LOCALAPPDATA%\Programs\Delivery Note Labeler\DeliveryNoteLabeler.exe
  - Or right-click PDF files in File Explorer -> Print Labels

Uninstall
  - Re-run Setup.exe, or extract the zip and run Uninstall.ps1

Settings
  - %USERPROFILE%\.delivery-note-labeler\config.json
